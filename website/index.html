为什么选择 HiKid？

开口就能聊

不需要打字，直接对着麦克风说话，AI 就能听懂并回复你。

隐私安全

对话、语音和模型全部在本地运行，不上传任何数据到云端。

无需联网

录音、识别、合成和对话全部通过本地流水线完成，没有网络也能用。

聪明又耐心

聊什么都行，说得慢或说得简单都没关系，AI 会耐心地陪你练习。

讲故事、玩游戏

简单三步，开始对话

1

下载 dmg

从 GitHub Releases 下载最新版本的 HiKid 安装包。

2

拖拽安装

打开 dmg 文件，将 HiKid 图标拖入 Applications 文件夹。

3

一键启动

第一次打开时，HiKid 会自动检测并下载所需依赖，无需手动配置。

首次启动 macOS 可能会阻止未签名应用。如遇提示，在终端运行 xattr -cr /Applications/HiKid.app 即可解决。

AI 后端配置

默认：Ollama 本地运行

HiKid 默认连接本地 Ollama 服务，使用 qwen3:0.6b 模型。所有对话完全在本地进行，无需 API Key。

ollama run qwen3:0.6b

自定义：其他 LLM 后端

你也可以在应用设置中配置自己的 LLM 后端，支持任何兼容 OpenAI API 的服务。

  • API Endpoint 地址
  • 模型名称
  • API Key
